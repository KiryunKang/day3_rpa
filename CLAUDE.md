     ## 프로젝트 작업 시 아래 지침을 따른다.
                                                                                                                                                                                                                                  ──
      ## 개발 환경 지침                                                                                                                                                                                                           ──
      - 코드 작성 시, 최대한 필수 코드만 작성하고 주석으로 설명한다.
      - 개발이 끝나면 반드시 테스트를 진행하고, 결과를 표 형식으로 보고한다.                                                                                                                                                      ──

        | 항목 | 명령 | 결과(통과/실패) | 비고 |                                                                                                                                                                                ──
        |---|---|---|---|
                                                                                                                                                                                                                              ──
      - 테스트 실패 시 임의로 고치지 말고, 원인과 함께 보고 후 지시를 기다린다.                                                                                                                                               ────
      - BE: uv 로 관리하는 Python 을 사용한다 (`uv run ...`).
      - FE: TypeScript + Vite + React 기반으로 작성한다.                                                                                                                                                                      ──

      ## 유의사항
      - 사용자가 지시하지 않은 내용은 억지로 추측하거나 추가하지 않는다.
      - 개발 전 정보가 부족하면 사용자에게 적극적으로 물어본다.

      ## 응답
       - 한국어로, 간결하게. 불필요한 서론·재요약 생략.
