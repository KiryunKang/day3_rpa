# skill creator

새 스킬을 만들 때 사용하는 스킬입니다. 아래 순서로 진행합니다.

## 1단계: 요구사항 확인
사용자에게 아래를 확인합니다. (이미 답이 나왔으면 건너뜀)
- 스킬 이름 (kebab-case, 예: `deploy-check`)
- 무슨 일을 하는 스킬인지 (한 줄 목적)
- 어떤 말을 하면 실행돼야 하는지 (트리거 키워드)

정보가 부족하면 추측하지 말고 사용자에게 물어봅니다.

## 2단계: 폴더/파일 생성
`.claude/skills/<이름>/SKILL.md` 를 만듭니다. 형식은 아래를 따릅니다.

```markdown
---
name: <kebab-case-이름>
description: <한 줄 목적>. 사용자가 '<트리거1>', '<트리거2>'라고 말하면 이 스킬을 참조한다.
---

# <이름>

<이 스킬이 하는 일 요약>

## 1단계: ...
## 2단계: ...

# 유의사항
- <지켜야 할 규칙>
```

- `description` 은 트리거 역할이므로 실행 키워드를 반드시 포함합니다.
- 본문은 실제 수행할 절차를 단계별로 적습니다.

## 3단계: 검토
생성한 SKILL.md 내용을 사용자에게 보여주고 수정할 부분이 없는지 확인합니다.

# 유의사항
- 스킬 이름은 소문자 kebab-case, 폴더명과 `name` 을 일치시킵니다.
- 응답과 스킬 내용은 한국어로 간결하게 작성합니다.
- 기존 스킬과 이름이 겹치지 않는지 먼저 확인합니다.
