개요

공공기관 행정 실무자의 반복 업무를 한 앱에서 처리하는 웹 슈퍼앱. 흩어진 도구(일정 공유, 엑셀 가공, 민원 응대, 뉴스 모니터링)를 통합해 업무 전환 비용을 줄인다.

👥대상

행정 담당자, 팀장, 민원 대응 담당자

🎯핵심 가치

반복 업무 자동화(RPA) · 정보 공유 · AI 보조

📦초기 범위

아래 4개 기능 + 기존 전자결재/공지 모듈

기능 요구사항 (PRD)

📅① 팀원 스케줄 관리

팀원들의 휴가·근무·출장 일정을 등록/공유하는 캘린더.

  • 必 일정 등록(제목/유형/기간/담당자/메모)
  • 必 월간·주간 캘린더 뷰, 유형별 색상
  • 必 수정/삭제
  • 選 담당자·유형·기간 필터

📊② 엑셀 업무자동화

특정 컬럼 기준으로 엑셀을 나누거나 여러 파일을 합친다.

  • 必 업로드 + 시트/컬럼 미리보기
  • 必 분할: 기준 컬럼값별 분리 → 다중시트/ZIP
  • 必 병합: 다중 파일 결합(헤더 검증)
  • 選 처리 이력 기록

💬③ 민원대응 챗봇

첨부 민원 매뉴얼 근거로 대응 방식·응대 스크립트 생성.

  • 必 매뉴얼 업로드(PDF/TXT/MD)
  • 必 매뉴얼 근거 응대 스크립트 + 인용(Citations)
  • 必 멀티턴 대화
  • 必 근거 없으면 "매뉴얼에 근거 없음" 고지

AI: Claude Opus 4.8 · 매뉴얼을 document 블록으로 전달 · 프롬프트 캐싱 · 스트리밍 응답

📰④ 뉴스 기사 수집

공공 행정 관련 뉴스를 매일 아침 자동 수집.

  • 必 매일 07:00 자동 수집(스케줄)
  • 必 RSS 등 소스 수집 + 키워드 필터
  • 必 URL 기준 중복 제거
  • 選 수동 수집 · AI 요약 브리핑

아키텍처

┌──────────────────────────┐ ┌──────────────────────────────┐ │ Browser (React SPA) │ /api │ FastAPI (uv, Python 3.14) │ │ 캘린더 / 엑셀 / 챗봇 │ ─────▶ │ 라우터: schedule / excel / │ │ / 뉴스 / 결재 │ 프록시 │ chatbot / news / documents │ └──────────────────────────┘ │ ├─ SQLite (app.db) │ │ ├─ APScheduler (매일 07:00) │ │ └─ Claude API (opus-4-8) │ └──────────────────────────────┘ │ │ 외부 뉴스 소스(RSS) Anthropic API

단일 FastAPI 앱에 기능별 APIRouter를 등록하는 모듈러 모놀리식. 백그라운드 스케줄러는 앱 lifespan에서 기동/종료.

│   └── src/
│       ├── api.ts, types.ts, App.tsx
│       └── features/      # schedule / excel / chatbot / news / approval

        ├── excel.py       # 엑셀 분할/병합
        ├── chatbot.py     # 민원 챗봇 (Claude API)
        ├── news.py        # 뉴스 수집 라우터
        ├── scheduler.py   # APScheduler 설정
        └── documents.py   # (기존) 전자결재

실행 방법

백엔드 (터미널 1)

cd backend
uv sync            # 최초 1회
uv run dev         # → http://127.0.0.1:8000  (문서: /docs)

프론트엔드 (터미널 2)

cd frontend
npm install        # 최초 1회
npm run dev        # → http://localhost:5173
민원 챗봇/AI 요약은 ANTHROPIC_API_KEY 환경변수가 필요하다. 키가 없어도 스케줄·엑셀·캘린더는 정상 동작한다.
